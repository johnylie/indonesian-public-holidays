Indonesian Public Holidays

A lightning-fast, highly-available JSON API and NPM Library for tracking national holidays and joint leaves (cuti bersama) in Indonesia. Use it as a standalone HTTP API or install it directly in your Node.js/TypeScript projects.

Blazing Fast

Leveraging Edge Caching on Vercel to deliver responses with minimal latency globally.

Always Accurate

Holidays are sourced directly from the latest official calendars, constantly updated.

Developer First

Simple REST endpoints returning standard JSON. Easy to integrate into any stack.

NPM Package

Also available as a library. Install via npm i indonesian-public-holidays to use in your JS/TS projects.

TypeScript Ready

Built with TypeScript. Includes full type definitions out of the box for a seamless developer experience.

Open Source

100% free and open source under the MIT License. Contributions and improvements are always welcome.
